# Usagi Pet / Usagi 桌宠

## English

Usagi Pet is a custom animated pet for Codex. It is a cream-yellow, long-eared, mischievous chibi bunny with simple hand-drawn lines, transparent background, and multiple small desktop-pet animation states.

This package includes:

- `pet.json` — Codex pet manifest
- `spritesheet.webp` — transparent 8x9 animation atlas

### Install

Copy this folder to your Codex pets directory:

```text
C:\Users\<you>\.codex\pets\usagi
```

Then open Codex and choose `Usagi` from:

```text
Settings -> Appearance -> Pet
```

If the pet does not appear immediately, reload the Codex window or restart Codex once.

### Animation States

The spritesheet includes idle, right/left movement, click/wave, happy jump, surprised/scream reaction, eating, excited wiggle, and study/review-style loops.

### Note

This is a fan-made custom Codex pet package for personal use. It is not an official Chiikawa asset or product.

## 中文

Usagi 桌宠是一个给 Codex 使用的自定义动画宠物。形象是奶黄色、长耳朵、古灵精怪的小兔子，使用简洁手绘线条、透明背景，并包含多种适合桌宠的小动画状态。

本仓库包含：

- `pet.json`：Codex 宠物清单文件
- `spritesheet.webp`：透明背景的 8x9 动画图集

### 安装方式

把这个文件夹复制到你的 Codex 宠物目录：

```text
C:\Users\<你的用户名>\.codex\pets\usagi
```

然后打开 Codex，在这里选择 `Usagi`：

```text
设置 -> 外观 -> 宠物
```

如果宠物没有立刻出现，可以重新加载 Codex 窗口，或完全退出后再打开一次。

### 动画状态

图集中包含待机、左右移动、点击/挥手、开心跳跃、惊讶/尖叫反应、吃东西、兴奋摇晃、假装学习/审阅等循环动作。

### 说明

这是一个用于个人使用的粉丝自制 Codex 桌宠包，不是官方 Chiikawa 素材或官方产品。
